# [에듀윌 화이트해커 양성과정] APT 악성코드 제작 (교육용 시뮬레이션)
![status](https://img.shields.io/badge/status-교육용--시뮬레이션-yellow) ![version](https://img.shields.io/badge/version-0.1.0-blue)

> 경고: 이 리포지토리는 교육 및 연구 목적의 모의 시뮬레이션용입니다. 이 저장소의 내용이나 도구를 실제 악의적 행위에 사용하지 마십시오.

## 설명 (Description)
이 프로젝트는 DDL(동적 라이브러리) 매뉴얼 매핑(Manual Mapping) 기법의 원리 이해와 실습을 위해 설계된 교육용 시뮬레이션입니다. 실제 악성 행위를 수행하는 코드를 제공하지 않으며, 알고리즘의 원리, 바이너리 패치/로딩 시나리오, 그리고 __HxD__ 같은 도구를 통한 분석 연습을 목적으로 합니다.

대상 독자: 다른 개발자 대상의 교육 자료

## 주요 기술 스택
- __Visual Studio 2022__ (C++14)
- HxD (바이너리 검사/편집 도구)
- libReaper (강사 제공 라이브러리, 시뮬레이션 API)

## 기능 (Features)
- DDL 매뉴얼 매핑 원리의 시뮬레이션 구현(교육용)
- 바이너리 로드/패치 시나리오 생성기(샘플 입력만 사용)
- libReaper 연동을 통한 인터페이스 호출 시뮬레이션
- 결과를 HxD 등으로 열어 분석할 수 있는 샘플 바이너리 출력
- 안전한 샌드박스/테스트 환경에서만 동작하도록 설계

## 설치 방법 (Installation)
1. 저장소 클론
git clone https://github.com/shpimit/manual_test.git cd manual_test
2. 사전 준비
- __Visual Studio 2022__를 설치하시고 C++ 데스크톱 개발 워크로드를 활성화하세요.
- 강사로부터 제공받은 `libReaper`를 `libs/libReaper` 폴더에 놓아주세요(실제 배포 라이브러리가 아닌 교육용 인터페이스).
- HxD 같은 바이너리 편집 도구를 설치하시길 권장합니다(분석용).
3. 솔루션 열기
- `manual_test\manual_test.vcxproj` 또는 `manual_test.sln` 파일을 __Visual Studio 2022__로 엽니다.
4. 빌드 설정
- 플랫폼 및 구성 (__x64__, __Debug__ 또는 __Release__)을 선택한 뒤, __Build > Build Solution__을 실행하세요.

참고: 외부 라이브러리 파일 경로는 프로젝트 설정의 `Additional Include Directories` 및 `Additional Library Directories`에 맞게 지정해야 합니다.

## 사용법 (Usage)
- 빌드하면 실행 파일 `manual_test.exe`(또는 설정한 출력 이름)가 생성됩니다.
- 프로그램은 시뮬레이션 입력을 받아 교육용 출력(샘플 패치 결과)을 생성합니다. 실제 공격 동작을 수행하지 않으니 안전한 테스트 파일을 사용하세요.

사용 시 주의:
- 생성된 바이너리나 결과물은 교육용 분석 목적으로만 사용하시고, 테스트는 격리된 가상머신 또는 샌드박스 환경에서 진행하세요.
- 실제 시스템에서 임의로 로딩/패치 작업을 실행하지 마십시오.

## 예제 파일
